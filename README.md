# HomeSec Explorer Video Guides

Companion guides for my YouTube tutorials.

This repository contains the **step-by-step guides** that accompany my videos on the  
[@HomeSecExplorer](https://www.youtube.com/@HomeSecExplorer) YouTube channel.
